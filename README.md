# unsplash-gmail-backgrounds

A browser extension that sets random images from Unsplash (a collection of free, beautiful images and photos) as your Gmail background using [Unsplash Source](source.unsplash.com). A new background image gets loaded into Gmail on each reload.

It fetches images that have 1080p resolution to ensure quality.

## how to use

**Chrome and Edge:** open the Extensions tab, turn Developer mode on, click the `Load unpacked` button and select the folder you downloaded the code into.

**Firefox:** open `about:debugging`, click the `This Firefox` button, then click the `Load Temporary Add-on...` button and select any file in the folder you downloaded the code into.
